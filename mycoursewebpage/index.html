AWS Databases

  • RDS
  • DynamoDB
  • Elasticache

Amazon RDS provides a way to set up relational database in the cloud. It also provides you with Database Migration Service to migrate your existing databases to RDS.

DynamoDB is a non relational DB supporting key-value and document data models. Its set up to support applications that scale globally holding upto millions of R/W requests per second.

Elasticache, popular choice for Gaming, IOT apps, is known to provide extreme performance improvements to existing apps by caching from in-memory data stores.
